# 🐶 Pet Management System

## 📖 About
This is my first C# console application.
The project demonstrates the fundamentals of Object-Oriented Programming (OOP) by creating a Pet class with attributes and methods.

## ✨ Features
- Create a Pet object
- Display pet information
- Pet can eat
- Pet can sleep
- Pet can play

## 🛠 Technologies Used
- C#
- .NET Framework
- Visual Studio 2022

## 📚 Concepts Practiced
- Classes
- Objects
- Methods
- Attributes
- Object creation
